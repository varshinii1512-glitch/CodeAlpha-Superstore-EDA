# Superstore Sales Data Analysis (EDA)

## Project Objective
To perform Exploratory Data Analysis (EDA) on the Superstore dataset in order to understand sales trends, profit performance, and overall business insights using Python.

## Dataset Information
- Total Records: 9,994
- Total Features: 21
- Missing Values: None
- Duplicate Records: None
- Dataset Type: Structured CSV data

## Tools & Technologies Used
- Python
- Pandas
- VS Code
- GitHub

## Analysis Performed
- Dataset inspection (shape, columns, data types)
- Summary statistics (describe)
- Missing value analysis
- Duplicate record detection
- Correlation analysis
- Region-wise Sales analysis
- Region-wise Profit analysis

## Key Business Insights
- Total Sales: 2,297,200.86
- Total Profit: 286,397.02
- Sales & Profit Correlation: 0.47 (Moderate Positive Relationship)
- The business is overall profitable
- Some regions contribute more significantly to total revenue and profit

## Business Understanding
- Higher sales generally lead to higher profit, but not always.
- Discounts impact profitability.
- Regional performance varies and can influence business strategy.

## Conclusion
The exploratory analysis reveals that the Superstore business is profitable overall, with moderate positive correlation between sales and profit. Regional performance differences highlight potential opportunities for strategic improvements.
